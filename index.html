Let's Get Started

  1. Open your terminal
    Mac: Spotlight → "Terminal"  |  Windows: search "Terminal"
  2. Install Claude Code
    npm install -g @anthropic-ai/claude-code
  3. Navigate to a project folder
    cd ~/Desktop/my-project
  4. Type claude and start talking
  5. Describe what you want to build — in plain English

Cheat Sheet

Terminal

pwd — where am I?
ls — what's here?
cd folder — go into folder
mkdir name — new folder
cd .. — go back up

Git

git init — start tracking
git add . — stage changes
git commit -m "msg" — save checkpoint
git push — back up to GitHub
git log — see history

Claude Code

claude — start session
/init — set up project
/clear — reset context
/help — all commands
/resume — pick up where you left off
Esc — stop / go back

Prompting

"Build me a..." — describe the goal
"Change X to Y" — be specific
"Explain this" — learn as you go
"Fix this error: ..." — paste errors
"Look at how X works" — explore first

Best Practices (from Anthropic's docs)

Give context — "I run a pottery studio and need a way to track inventory"
Start small, then iterate — get one thing working before adding complexity
Correct, don't restart — tell Claude what's wrong; it learns from feedback
Give it a way to verify — "run the tests" or "take a screenshot and check"
Use /clear between tasks — fresh context = better results
Commit early and often — git is your undo button; don't be afraid to break things
Explore before building — ask "how does X work?" before "build me X"
Point to examples — "follow the pattern in this file" beats a long description
